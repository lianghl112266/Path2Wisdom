# 题目描述

**Time Limit:** 1000 ms  
**Memory Limit:** 256 mb

实现一个加法器，使其能够输出 `a + b` 的值。

## 输入输出格式

### 输入描述:
输入包括两个数 `a` 和 `b`，其中 `a` 和 `b` 的位数不超过 1000 位。

### 输出描述:
可能有多组测试数据，对于每组数据，输出 `a + b` 的值。

## 输入输出样例

### 输入样例:
```
2 6
10000000000000000000 10000000000000000000000000000000
```

### 输出样例:
```
8
10000000000010000000000000000000
```
